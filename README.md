### description
It use [gulp](http://gulpjs.com) to compress , minify , build sys.
( The html's template is [handlebars](http://handlebarsjs.com) )

+ gulp
+ gulp-compass
+ gulp-concat
+ gulp-imagemin
+ gulp-minify-css
+ gulp-minify-html
+ gulp-plumber
+ gulp-rename
+ gulp-uglify
+ gulp-compile-handlebars

### command

build sass , js , handlebars , then watch
```
$ gulp
``` 

minify css , js
```
$ gulp minify-css
$ gulp minify-js
```


concate **js** or **css** file to one file ,  reduce request
```
$ gulp concate-css
$ gulp concate-js
```

compress img 
```
$ gulp image
```
